Packet Analyzer - Python

This Python project analyzes network packets from a `.pcap` file.

Features
- Counts total packets
- Counts TCP, UDP, and ICMP packets
- Identifies the most common source IP
- Saves results to a text file (`analysis_result.txt`)

How to Run
1. Upload your `.pcap` file to the same directory as the script.
2. Run `network_analyzer.py` using Python 3.
3. Check `analysis_result.txt` for results.

Purpose
- Learn network packet analysis
- Practice working with real network data
- Understand packet distribution and source IP patterns
