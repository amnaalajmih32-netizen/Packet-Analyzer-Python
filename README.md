# Packet Analyzer - Python

This is an educational Python project that analyzes network packets from a `.pcap` file. 

## Features
- Counts total packets
- Counts TCP, UDP, and ICMP packets
- Identifies the most common source IP
- Saves results to a text file

## How to run
1. Upload your `.pcap` file to the same directory
2. Run `network_analyzer.py` using Python 3
3. Check `analysis_result.txt` for results
